# 🤖 Jarvis AI Chatbot

An advanced AI-powered chatbot built using **Streamlit**, **Groq API**, and **MongoDB** with a modern UI and multi-chat support.

---

## 🚀 Features

✨ Multi-chat system (like ChatGPT)  
📁 File upload support (PDF, CSV, images, text)  
🧠 AI responses using Groq (LLaMA model)  
💬 Chat history with switching between conversations  
🗑️ Delete chats functionality  
🔐 User authentication (Login & Signup)  
🎨 Animated UI with modern design  
🌊 Dynamic background effects  

---

## 🛠️ Tech Stack

- **Frontend:** Streamlit  
- **Backend:** Python  
- **AI Model:** Groq API (LLaMA 3)  
- **Database:** MongoDB  
- **Authentication:** bcrypt
